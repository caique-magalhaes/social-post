# Social Post

Social Post is a full-stack, single-page social media application. It allows users to create posts, follow profiles, interact with content via asynchronous likes, and edit posts without reloading the page.

## 🛠️ Tech Stack
* **Backend:** Python, Django
* **Frontend:** JavaScript (Asynchronous ES6+), HTML5, Custom CSS3
* **Database:** SQLite

## 🌟 Key Features
* **Dynamic Social Feed:** Features like "Following" feeds with responsive pagination.
* **Asynchronous Interactions:** Implements JavaScript `fetch()` APIs to toggle likes and update post content instantly without full-page refreshes.
* **Following Page Customization:** The following page allows you to see only the content from profiles you are following.
